Ejercicio 1

Convertir monedas en tiempo real.

En este ejercicio, puedes ingresar cualquier tipo de moneda (de otros países) e instantáneamente se convertirán a lempiras.

Ejercicio 2

Área de un círculo

Este ejercicio calcula el área de un círculo, solicitando el radio como dato de entrada.

Ejercicio 3

Perímetro de un círculo

Este ejercicio calcula el perímetro de un círculo con el valor del radio proporcionado por el usuario en el codigo.

Ejercicio 4

Mayor de edad o menor de edad

Este ejercicio determina si una persona es mayor o menor de edad basándose en su edad.

Ejercicio 5

Promedio de cuatro notas

Este ejercicio calcula el promedio de cuatro notas ingresadas por el usuario.

Ejercicio 6

Rango de promedio

Este ejercicio clasifica el promedio ingresado en un rango, determinando si es avanzado, debe mejorar o bajo.

Ejercicio 7

Venta de boletos

Este ejercicio simula la venta de un boleto, y se nos da un descuento depende nuestra condicion (edad, estudiante).

Ejercicio 8

Conversión de metros a centímetros y viceversa

Este ejercicio convierte una cantidad de metros ingresada por el usuario a centímetros o en viceversa.

Ejercicio 9

Área del cuadrado

Este ejercicio calcula el área de un cuadrado dado el valor de un lado.

Ejercicio 10

Área de un trapecio

Este ejercicio calcula el área de un trapecio con base en la longitud de las bases y la altura insertadas por el usuario ya en el codigo.

Ejercicio 11

Área de un triángulo

Este ejercicio permite calcular el área de un triángulo segun los valores ingresados de la base y la altura.

Ejercicio 12

Conversión de Celsius a Fahrenheit

Este ejercicio convierte una temperatura ingresada en grados Celsius a Fahrenheit.

Ejercicio 13

Conversión de dólares a lempiras

Este ejercicio convierte una cantidad de dólares ingresada a lempiras.

Ejercicio 14

Lempiras a dólares

Este ejercicio convierte una cantidad de lempiras ingresada a dólares.

Ejercicio 15

Licencia de conducir

Este ejercicio te pide ingresar tu edad, y te dice si estas apto para conducir..
